This project contains a Xamarin app that calls into a simple decision service to do a basic operation with each of the supported
datatypes.

Its goal is to illustrate how to call Javascript from C#, marshal data in and out of the decision service.

It does support Decimal as this is a complex type.  DateTime to be added later.

Prerequisites:

You will need to install the following:
  _ Visual Studio install
  _ Xamarin tools
  _ Android SDK
